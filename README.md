### 开发前的配置要求

1. easyx
2. VS2022
3. 项目属性中关闭SDL检查，修改字符集为多字符集

### 安装步骤

1. 从项目压缩文件中解压缩
2. .sln文件导入VS2022
3. 运行main.cpp

1. 通过GitHub克隆到本地目录`git clone https://github.com/Andrew211vibe/FireMiaoMiao.git`

或者:

1. 运行安装包安装
